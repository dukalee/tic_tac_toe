# Project Title

Tic Tac Toe Program

## Program Description 

Made for teaching python programming. 
Simple, very well-known 2-player game 

## How to Play 

Player 1 : O 
Player 2 : X 

At first, one player is chosen at random to go first. 
Player who places three in a row (row, column, diagonal) first wins the game.
